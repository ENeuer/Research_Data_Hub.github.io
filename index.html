About this Website

In 2024 I started changing my classical PDF portfolio to a web based format, leveraging the web development skills I once gained in high school. Enriched by web development courses, AI tools and UX knowledge, I was able to code and design this entire website from scratch in approx. 2 months.

This process has been fun, educational and insightful, but I am far away from any higher level of expertise. Both in UX design and web development.
Feel free to reach out to find out more about my process or have a look at this website's Github repository.
